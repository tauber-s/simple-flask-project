# Flask + Pytest Example

Simple project using Flask with tests via Pytest.

## How to use

1. Virtual env
```bash
python3 -m venv venv
```

2. Active venv
```bash
source venv/bin/activate
```

3. Install requirements
```bash
pip install -r requirements.txt
```

4. Run flask
```bash
flask --app app run --host=0.0.0.0
```

5. Run tests
```bash
pytest -v
```
